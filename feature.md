# A-DXC Intranet Feature Design

## Overview
ชุดฟีเจอร์สำหรับ Intranet / Dashboard ของ A-DXC ที่เน้น:
- การสื่อสารภายในองค์กร (Internal Communication)
- การเข้าถึงทีม IT ได้ง่ายขึ้น
- การสร้าง Culture และ Engagement ภายใน

---

# 1. Feature: A-DXC Announcements & Security

## Objective
ใช้สำหรับประกาศข่าวสารสำคัญ และแจ้งเตือนด้านความปลอดภัยให้พนักงานรับรู้แบบรวดเร็ว

## Function
- แสดงประกาศทั่วไป (Announcements)
- แสดง Security Alert (เช่น Phishing, Malware)
- Highlight ข่าวสำคัญ (Pinned / Urgent)
- รองรับการแยกประเภทข่าว

## Content Types
- Announcement
- Security Alert
- New Tools / Software Release

## Example Use Cases
- แจ้ง downtime / maintenance server
- แจ้งเตือนอีเมล phishing
- เปิดตัวระบบใหม่ในองค์กร

---

# 2. Feature: Meet the IT Team & IT Tips of the Week

## Objective
- ลดช่องว่างระหว่างทีม IT กับพนักงาน
- ทำให้ IT เข้าถึงง่าย เป็นมิตร
- แชร์ความรู้ด้านเทคโนโลยีแบบง่าย ๆ

## Function
- แสดงทีม IT พร้อมรายละเอียด
- แสดง Tips รายสัปดาห์

---

# 3. Feature: A-DXC Activity

## Objective
- เก็บภาพกิจกรรมขององค์กร
- ใช้เป็น reference ในอนาคต
- สร้างความรู้สึกมีส่วนร่วม
